# 🧠 Smart City Traffic Optimization

## 📌 Project Overview
This project analyzes and predicts urban traffic volume using time, weather, and holiday features. The goal is to support data-driven traffic management strategies aligned with smart city mobility initiatives.

---

## 📊 Dataset
- Source: Kaggle – Metro Interstate Traffic Volume
- Records: ~48,000
- Target Variable: traffic_volume

---

## 🔎 Key Insights
- Traffic peaks during 7–9 AM and 4–6 PM
- Traffic decreases during public holidays
- Severe weather slightly reduces traffic volume

---

## 🤖 Machine Learning Models

### 1️⃣ Linear Regression
- R² Score: 0.16
- MAE: ~1,600 vehicles

### 2️⃣ Random Forest (Final Model)
- R² Score: 0.95
- MAE: 234 vehicles

The Random Forest model explains 95% of traffic variability and provides highly accurate predictions.
